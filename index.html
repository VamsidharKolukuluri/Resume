Stress Detection through Speech Analysis

  • A project was built to develop a model that can detect stress in speech using a convolutional neural network (CNN).
  • The RAVDESS dataset was used to train the model, and Mel-frequency cepstral coefficients (MFCCs) were used as a preprocessing technique
  • The CNN was able to achieve an accuracy of 89.16% on the test set.
  • The results suggest that CNNs can be used to effectively detect stress in speech, which could be a valuable tool for a variety of applications.
